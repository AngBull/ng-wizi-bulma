by

node_modules/bulma-calendar/dist/js/bulma-calendar.js

The datepicker allows users to enter a date by choosing a date from the calendar.

Add bulma-calendar.js

In angular.json file at the root of your folder, add the following line in project > architect > build > options > scripts "node_modules/bulma-extensions/bulma-calendar/dist/js/bulma-calendar.js". This will allow you to import the modules from Bulma Calendar.

The official documentation is available at the following link https://creativebulma.net/product/calendar/demo

Format and Internalization

Internationalization of the datepicker is configured via four aspects.

Here is the list of the basic options of the NwbDatePickerFormat, in the file date-picker-format.ts:


By default the NwbDatePickerFormat will creates dates in your time zone specific locale. You can change the default behaviour to parse dates as UTC by providing the NWB_DATE_LOCALE and setting it to useValue with lang as you want.


It is also possible to create your own DateAdapter that works with any date format your app requires. This is accomplished by subclassing DateAdapter and providing your subclass as the DateAdapter implementation. You will also want to make sure that the NWB_DATE_FORMAT provided in your app are formats that can be understood by your date implementation. See Customizing the parse and display formats for more information about NWB_DATE_FORMAT.
